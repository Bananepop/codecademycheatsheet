Selector Description Syntax Examples
Type Selects all elements of a given type throughout the HTML document. type {property: value;} p {color:green}
Universal Selects all elements of any type. It uses the * character where you specified the type selector in a ruleset. * {property: value;} * {font-family: Verdana;}
Class A Class is like a category. When we put classes to HTML elements we are categorizing them.
To select an HTML element by its class using CSS, a period (.) must be prepended to the class' name.		 . class_name {property: value;} HTML:<p>class="brand">Sole Shoe Company</p>
CSS: .brand {color: pink}
Multiple Classes The concept of multiple classes in CSS refers to the possibility of assigning more than one class to an HTML element.
This allows you to apply several different sets of styles to the same element. With a class you can set several properties (styles) to the same element. With multiple classes you can create different sets of properties and apply them to the same element when needed without having to create a separate class for each combination.		 . class_name1 {property: value;}

. class_name2 {property: value;} 		HTML:<p>class="class1 class2">paragraph with multiple classes</p>
CSS: .class1 {color: pink;}
CSS: .class2 {background-color: yellow;}
ID (Identifier) A unique identifier for an HTML element, specified using the id attribute, and is used in CSS to apply styles to that specific element. To select an element's ID with CSS we prepend the ID name with a number sign (#) #id_name {} HTML:<h1>id="main_title">Fit Journal</h1>
CSS: #main_title {}
Attribute An attribute selector is a way to target HTML elements based on the presence or value of their attributes. This alleviates the need to add new code like the Class or ID attributes.
Remember that Attributes are content added to the opening tag of an element and are made up of two parts: Name=Value		 The most basic syntax is an attribute surrounded by square brackets:

[href]{color: magenta;}

It can get more granular from there by adding type and/or attribute values:

type[attribute*=value]		 We have the following attribute in HTML:

<img src='images/seasons/cold/winter.jpg'>

To select it in CSS we have to:

img[src*='winter']{height: 50px;}
img is the type
src is the attribute
winter is the value
This ruleset looks for an element (img) with an attribute of src that contains the string 'winter', and sets the height to 50px

Specificity Specificity is the order by which the browser decides which CSS styles will be displayed.

To make styles easy to edit it's best to style with a type selector if possible. If not, add a class selector. If that is not specific enough, then consider using an ID selector.		 IDs are the most specific selectors in CSS, followed by classes, and finally, type. HTML: <h1 class='headline'>

CSS: h1 {color: red;}

.headline {color: firebrick;}

In the example above, the color of the headline would be set to firebrick as class selectors are more specific than the type selector
Chaining CSS selectors can be chained so that rule sets apply only to elements that match all criteria. For instance, to select <h3> elements that also have the section-heading class, the selector h3.section-heading can be used. /* Select h3 elements with the section-heading class */

h3.section-heading { color: blue;}

/* Select elements with the section-heading and button class */

.section-heading.button { cursor: pointer;} 		Best examples

Multiple Selectors Grouping CSS selectors allows you to apply the same styles to multiple elements efficiently by using a single rule separated by commas. The comma in CSS selectors functions as "and," applying styles to multiple elements collectively. If the comma were missing, the selector would instead apply to all paragraph elements that are a child of a division. That is a different kind of selector, so the comma is important. selector, selector2, selector3...{property:value;} p, .red, #sub, div a:link { color: #f00; }

This CSS rule would apply to:

* Any paragraph element
* Any element with the class of red
* Any element with an ID of sub
* The link pseudo class of the anchor elements that are descendants of a division.

That last selector is a compound selector. As shown, it's easily combined with the other selectors in this CSS rule. The rule sets the color of #f00 (red) on these four selectors, which is preferable to writing four separate selectors to achieve the same result.
